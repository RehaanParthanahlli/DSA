STAGE 1: FREQUENCY PROBLEMS
================================================

================================================
prefix[i] = prefix[i-1] + arr[i]
- Subarray sum = k
- Prefix XOR equals
- Cumulative frequency

================================================
STAGE 3: SLIDING WINDOW HASH
================================================
Maintain hashmap in O(1)
- No-repeat substring
- Character replacement
- Fixed window sums

================================================
STAGE 4: COMPLEX KEYS
================================================
Hashable keys (tuple, str, frozenset)
- Graph nodes as tuples
- Accounts as frozenset(emails)
- Division as (a,b) fractions

STAGE 5: HASH + DATA STRUCTURES
================================================
HashMap + List/Heap/DLL
- LRU/LFU Cache

- Priority access

STAGE 6: ROLLING HASH (STRINGS)
================================================
Hash(string[i:j]) in O(1)
- Rabin-Karp substring
- Longest repeated substring
